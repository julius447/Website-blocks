# Website-blocks

Reusable, production-bound web blocks for **ampy.se** — each category in its own folder,
built on the real Ampy design tokens and delivered Bricks/FluentSnippets-ready.

## Categories

- **[Header](Header/)** — header / navigation redesign. Single clean floating bar
  (Google/phone top-bar removed), 3 mega-menus on the commercial tiers
  (Tjänster · Produkter · Lösningar), pulse "Boka rådgivning" CTA, page-dim overlay
  on menu-open, scroll-tighten, and a right-side mobile drawer with tap-to-expand
  accordions. Synthesised from HemSol (scroll + pulse) + EVFi (mega-menu IA).
  Prototype — leaf links are placeholders pending WP menus 29–32.
